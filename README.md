# Api endpoints:

Localhost:8000/

In use

`<b>`GET`</b>`

- Status `</br>`
  Returns every book's status
- Status/id `</br>`
  Returns book information by book id
- Books `</br>`
  Returns every book's information

`<b>`POST`</b>`

- Loan `</br>`
  Updates book's loan status (True or False)
